# python-mini-projects

Weekly Python mini-projects built from scratch using PyCharm.
Each project is self-contained and written to PCPP1 standard —
type hints, docstrings, exception handling, and PEP 8 throughout.

Each project folder contains:
- A detailed description (what we are building, inputs, outputs, rules)
- The final polished version only — no drafts
